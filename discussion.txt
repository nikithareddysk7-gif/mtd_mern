        Region: Mumbai ap-south-1
        -> Create Deployment
        !Under Deployment 
     4. Side Menu Bar-> Clusters 
        If "Connect" is there 
        we may connect from our computer
        Need: 
            1. username 
            2. network inbound connection 
     5. Under Security in Left Side Menu Bar 
        -> Database and Network Access 
        Username: mahesh
        Password: 1234
        Built-in-Role:Atlas Admin 
        -> Add User  
    6. Under Security in Left Side Menu Bar 
        -> Database and Network Access
        -> IP Access List
        -> Add IP Access List Entry 
           Access List Entry: 0.0.0.0/0
           (Everyone from everywhere) 
        -> Confirm 
    7. Ready to connect to cloud mongo server 
    7.1. Connect via Mongo Shell 
        Side Menu Bar 
        -> Clusters 
        -> Connect

            -> I dont have Mongo Shell 
            -> Download the shell 
            -> Install shell 
            -> Terminal or Command Prompt 
            $ mongosh --version
            It displayed the version. 
        -> Copy connection string 
            mongosh "mongodb+srv://cluster0.we7nuaw.mongodb.net/" --apiVersion 1 --username mahesh
        -> Extract only the connection string 
           !Modify it 
           mongodb+srv://cluster0.we7nuaw.mongodb.net/

           mongodb+srv://username:password@cluster0.we7nuaw.mongodb.net/

           -> 
           mongodb+srv://mahesh:1234@cluster0.we7nuaw.mongodb.net/

           -> 
           mongosh "connectionString"
